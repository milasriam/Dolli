# Project Summary
This project is a micro-donation platform designed to enhance user engagement in charitable activities. It features campaign management, donation facilitation, and user profiles, creating a seamless experience for users to participate in campaigns and contribute to causes. The application empowers users to create, manage, and track their campaigns while providing insights into their impact and achievements.

# Project Module Description
The project consists of multiple functional modules:
- **Authentication**: Manages user login and authorization.
- **Campaign Management**: Handles the creation, updating, and tracking of campaigns.
- **Donations**: Facilitates user donations and tracks their history.
- **User Profiles**: Allows users to manage their profiles and view their activities.
- **Analytics**: Provides insights into user engagement and campaign performance.
- **Badges**: Manages user achievements and rewards.

# File Description Inventory
- **app/backend/main.py**: Entry point for the backend server.
- **app/frontend/src/App.tsx**: Main application component for the frontend.
- **app/frontend/src/contexts/AuthContext.tsx**: Provides authentication context for the application.
- **app/frontend/src/pages/CreateCampaign.tsx**: Page for creating new campaigns.
- **app/frontend/src/pages/Profile.tsx**: User profile page displaying user statistics and activities.
- **app/backend/routers/**: Contains route definitions for API endpoints.
- **app/frontend/components/**: Contains reusable UI components.
- **app/backend/data_models/**: JSON files defining the structure of data for badges, campaigns, etc.
- **app/backend/models/**: Python files defining the ORM models for badges, campaigns, donations, and referrals.

# Technology Stack
- **Frontend**: React, TypeScript, Vite
- **Backend**: Python, FastAPI, SQLAlchemy
- **Database**: PostgreSQL
- **Deployment**: Docker, AWS

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   cd /workspace/app/frontend
   pnpm install
   cd /workspace/app/backend
   pip install -r requirements.txt
   ```
2. Build the frontend:
   ```bash
   cd /workspace/app/frontend
   pnpm run build
   ```
3. Run the backend server:
   ```bash
   cd /workspace/app/backend
   uvicorn main:app --reload
   ```
